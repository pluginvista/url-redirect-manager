# URL Redirect Manager

URL Redirect Manager is a lightweight and easy-to-use WordPress plugin that helps you create, manage, and track URL redirects directly from the WordPress admin panel.

It supports redirect history tracking, bulk redirect operations, and secure AJAX-based actions.
